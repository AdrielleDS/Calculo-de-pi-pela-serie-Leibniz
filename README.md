# Calculo-de-pi-pela-serie-Leibniz

Os resultados dos testes  realizados para o cálculo do 
número de , utilizando a Serie de Leibniz. O número  é uma constante 
π
matemática fundamental amplamente utilizada em diversas áreas da ciência e da 
engenharia. Uma das formas de aproximar seu valor é por meio da Série de Leibniz, 
uma série infinita alternada que, embora tenha convergência lenta, é bastante 
utilizada em estudos computacionais. O presente trabalho tem como objetivo 
implementar o cálculo aproximado do número  utilizando a Série de Leibniz por 
π
meio de programação paralela com a biblioteca Pthreads na linguagem C++. O 
programa foi desenvolvido em ambiente Linux por meio do VSCode Online (GitHub 
Codespaces) e executado utilizando diferentes quantidades de threads (1, 2, 4 e 8), 
realizando cinco testes para cada configuração. Os tempos de execução foram 
medidos com o comando time e em seguida, foi calculada a média dos tempos 
obtidos. Os resultados permitiram analisar o impacto do paralelismo no desempenho 
do programa, evidenciando a redução do tempo de execução com o aumento do 
número de threads, até o limite dos recursos disponíveis. Dessa forma, o trabalho 
demonstra a eficiência da programação paralela aplicada ao cálculo numérico do 
número .
